zds_action_tile

int zds_action_tile(zds_hdlg hDialog,char *szControlKey,void *pfnControlFunc);

Assigns an action to evaluate when the user selects the specified tile.

The szControlKey argument is the name of the tile that triggers the action (specified as its key attribute). The action assigned by zds_action_tile() supersedes the dialog box's default action (assigned by zds_new_dialog()) or the tile's action attribute, if these are specified.

The pfnControlFunc argument is a pointer to a function that is called when the tile is selected. The hDialog argument specifies the dialog box. The callback function is passed a callback packet, as follows:

typedef struct {

   zds_hdlg dialog; /* Handle of the dialog box */

   zds_htile tile; /* Handle of the tile */

   char *value; /* Current value of the tile */

   void *client_data; /* Current client data */

   int reason; /* Callback reason code */

   long x, y; /* Image button coordinates */

 }

zds_callback_packet;

The function returns RTNORM if successful; otherwise, it returns an error code.

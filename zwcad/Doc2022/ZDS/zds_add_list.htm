zds_add_list

int zds_add_list(char *szAddToList);

Adds the specified string to the current list (or changes a list item).

Adds the string specified by szAddToList to the currently active list, or replaces a current list item with szAddToList. (You had to open the list with zds_start_list().) It returns RTNORM if successful; otherwise, it returns an error code.

zds_client_data_tile

int zds_client_data_tile(zds_hdlg hDialog, char *szControlKey, void *pApplicationData);

Associates application-managed data with the specified tile.

Associates application-managed data with the tile specified by szControlKey. The data is specified in the pApplicationData argument. The hDialog argument specifies the dialog box. Returns RTNORM or another error code.

Data can be passed to and from callbacks in pApplicationData, avoiding the side effects that would occur if global variables were used instead.

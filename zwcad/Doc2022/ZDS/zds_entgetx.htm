zds_entgetx

struct zds_resbuf *zds_entgetx (const zds_name entity, const struct zds_resbuf *applist);

Get information and EED about this object.

This function is very similar to zds_entget, except that the result buffer list returned by zds_entgetx includes the EED (an acronym for extended entity data, also referred to as xdata) for the application specified in applist.

The argument entity represents the name of the object whose information you wish to retrieve. The argument applist represents a linked list of application names, whose extended entity data you want included in the result buffer returned by this function.

This function returns the result buffer list with the object's information and extended entity data, or null if the object is not found.

Example

zds_name entname1;

struct zds_resbuf *applist;

struct zds_resbuf *entlist;

applist=zds_buildlist(RTSTR,"myapp",RTNONE);

entlist=zds_entgetx(entname1,applist);

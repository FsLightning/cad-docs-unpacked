zds_exit

void zds_exit (int status);

Notify ZWCAD that the ZDS application is exiting now, then exit.

This function tells ZWCAD that the ZDS application is about to exit and then exits. If the application exits without telling ZWCAD, the memory and resources it uses are not freed.

The status argument represents the way in which the application terminated. Use 0 (zero) for a planned and intentional termination. Other values can be used for unusual reasons for exiting the application, depending on your system.

This function returns void.

Example

zds_printf("\nOURBASE is shutting down now.");

zds_exit(0);

zds_get_attr

int zds_get_attr(zds_hdlg hDialog, char *szControlKey, char *szAttribName, char *szAttribInfo, int lInfoMaxLength);

Retrieves an attribute's dialog control language (DCL) value.

The szControlKey argument, which is case sensitive, specifies the tile, and szAttribName specifies the name of the attribute as it appears in the tile's DCL description. The value returned is the attribute's initial value as specified in its DCL description; it does not reflect changes to the state of the tile that come about through user input or zds_set_tile() calls.

The szAttribInfo argument is set to the attribute value. The application must allocate space for the szAttribInfo string and pass its length in the lInfoMaxLength argument. The lInfoMaxLength argument should include room for the string terminator (\0 or EOS). The hDialog argument specifies the dialog box. The zds_get_attr() function returns RTNORM if it succeeds; it returns RTERROR if it can't find the specified tile or attribute.

The function returns RTNORM if successful; otherwise, it returns an error code.

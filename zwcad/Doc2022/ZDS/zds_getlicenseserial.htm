zds_getlincenseserial

int zds_getlicenseserial (int* rettype, char* serial, int chklock);

Return the license status and license code.

This function return the current license status and serial number.

The parameter rettype returns the license status to user, the value representative no license (0), software encryption(1), hardware encryption (2).

The parameter serial returns the license code to user.

The parameter chklock gets input from user, check the lock if this parameter is not 0.

This function returns ZDS_RTNORM if all right, returns ZDS_RTERROR otherwise.

Example

int retType = -1;

char Serial[256];

if (ZDS_RTNORM == zds_getlicenseserial(&retType, Serial, 0))

{

     zds_printf("The license type is %d. \nThe license code is %s", retType, Serial);

     return true;

}

else

{

     return false;

}

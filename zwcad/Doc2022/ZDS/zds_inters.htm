zds_inters

int zds_inters (const zds_point point1, const zds_point point2, const zds_point point3, const zds_point point4, int mode, zds_point interspt);

Find where these lines intersect.

This function finds the intersection point of two lines.

The arguments point1 and point2 represent the starting and ending points for the first line segment, while point3 and point4 are the starting and ending points for the second line segment.

The mode argument controls whether the line segments must actually cross:

Mode Meaning
0 The lines do not need to cross.
1 The lines must cross.

When mode is 1 and the line segments do not intersect, zds_inters returns RTERROR.

The interspt argument is the point at which the two lines intersect.

This function returns RTERROR or RTNORM.

Example

int cross;

cross = 1;

zds_point firstpt,secondpt,thirdpt,fourthpt,interspt;

zds_getpoint(NULL,"\nFirst point for line 1: ",firstpt);

zds_getpoint(firstpt,"\nSecond point: ",secondpt);

zds_getpoint(NULL,"\nFirst point for line 2: ",thirdpt);

zds_getpoint(thirdpt,"\nSecond point: ",fourthpt);

returni=zds_inters(firstpt,secondpt,thirdpt,fourthpt,cross,intpt);

if (returni != 5100)

zds_printf("\nThese two line segments do not intersect");

if (returni == 5100)

zds_printf("\nIntersection: (%f,%f,%f)", intpt[0],intpt[1],intpt[2]);

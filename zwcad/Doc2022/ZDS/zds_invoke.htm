zds_invoke

int zds_invoke (const struct zds_resbuf *args, struct zds_resbuf **answer);

Call a function in another ZDS application.

This function calls a function defined in another ZDS application.

The args argument is a result buffer list that contains the external function call.

The answer argument contains the external function's return values.

This function returns RTERROR or RTNORM.

Note Make sure that external functions are registered by zds_regfunc.

zds_iscntrl

int zds_iscntrl (int val);

Determine whether this is a control character.

The function determines whether a given character is a control character. The argument val is the ASCII value for a character.

This function returns non-zero if the given character is alphabetical. Otherwise, it returns zero.

Example

int returni;

char string1[512];

zds_getstring(1,"\nEnter a control character: ",string1);

returni=zds_iscntrl(string1[0]);

if (returni == 0)

zds_printf("\nThe character is NOT a control character. ");

else

zds_printf("\nThe character IS a control character. ");

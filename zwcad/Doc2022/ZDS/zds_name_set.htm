zds_name_set

int zds_name_set (zds_name name1, zds_name name2);

Copy n entity name or selection set to another.

This function copies an entity name (or a selection set name) to another name.

The argument name1 is the name of originating entity or selection set, while name2 is the target name to which to copy.

zds_new_positioned_dialog

int zds_new_positioned_dialog(char *szDialogName, int nDialogFileID, void *pfnDef_Callback, int nXPos, int nYPos, zds_hdlg *hDialog);

Initializes a dialog box at a specified screen position.

Initializes a dialog box at the screen position specified by (nXPos, nYPos). The nXPos and nYPos arguments usually specify the upper-left corner of the dialog box, but this is platform dependent, as is the system of units in which the location is specified.

To ensure the portability of your application, you should not specify these coordinates the first time you initialize the dialog box. The coordinates are meant to be retrieved from a previous call to zds_done_positioned_dialog() and are to be used when your application reopens the dialog box. On platforms that allow the user to move a dialog box, this function enables you to reopen it in the same position in which the user left it. If you pass both nXPos and nYPos as -1, the dialog box is opened in the default position (the center of the ZWCAD graphics screen).

Except for the coordinates, zds_new_positioned_dialog() behaves exactly like zds_new_dialog().

The function zds_new_positioned_dialog returns RTNORM if successful; otherwise, it returns an error code.

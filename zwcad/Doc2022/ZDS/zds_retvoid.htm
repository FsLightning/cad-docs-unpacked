zds_retvoid

int zds_retvoid (void);

Return void to LISP.

This function returns nothing to LISP. Since LISP only returns the last value sent to it, you can use zds_retvoid to cause LISP to return nothing. Use this as the last function to keep from printing what would have otherwise been the last function.

This function returns just RTNORM.

Example

zds_retvoid();

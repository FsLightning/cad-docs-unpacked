zds_set_tile

int zds_set_tile(zds_hdlg hDialog, char *szControlKey, char *szControlInfo);

Sets the value of a tile (initially set by the value attribute).

The szControlKey argument specifies the tile, and szControlInfo is the new value to assign. The hDialog argument specifies the dialog box.

The function returns RTNORM if successful; otherwise, it returns an error code.

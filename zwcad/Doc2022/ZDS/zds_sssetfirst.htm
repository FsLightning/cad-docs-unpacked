zds_sssetfirst

int zds_sssetfirst (const zds_name grips, const zds_name select);

Highlight and/or grip these objects.

This function causes objects in a drawing to be gripped and/or highlighted (selected). Grips are the small selection squares that surround an object.

The grips argument is a selection set of objects to be gripped in the drawing.

The select argument is a selection set of objects to be selected in the drawing.

This function returns RTNORM or RTREJ.

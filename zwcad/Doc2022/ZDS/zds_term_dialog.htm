zds_term_dialog

int zds_term_dialog(void);

Terminates all current dialog boxes as if the user had canceled each of them.

If an application terminates while any DCL files are open, ZWCAD automatically calls zds_term_dialog().

This function is used mainly for aborting nested dialog boxes.

The zds_term_dialog() function always returns RTNORM.

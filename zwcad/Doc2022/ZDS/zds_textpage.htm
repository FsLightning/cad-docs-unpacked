zds_textpage

int zds_textpage (void);

Switch to the cleared text window.

This function switches to the text window from the graphics window. This functions differs from zds_textscr in that this function clears the text screen first.

This function returns RTNORM or RTERROR.

Example

char string1[512];

zds_getstring(0,"\nTo switch to the text screen, press Enter. ",string1);

zds_textpage();

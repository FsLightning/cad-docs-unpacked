zds_tolower

int zds_tolower (int value);

Change this character to lower-case.

This function converts a given character from upper- to lower-case.

The argument value is the ASCII value of a character.

This function returns the ASCII value for its lower case counterpart when the character is upper-case. If the given character is lower-case already, zds_tolower leaves it as it is.

Example

int returni;

char string1[512];

zds_getstring(1,"\nEnter an upper case character: ");

returni=zds_tolower(string1[0]);

zds_printf("\nzds_tolower returned %i",returni);

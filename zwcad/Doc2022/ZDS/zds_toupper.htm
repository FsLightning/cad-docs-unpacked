zds_toupper

int zds_toupper (int value);

Change this character to upper-case.

This command converts a given character from lower- to upper-case.

The argument value is the ASCII value of a character.

This function returns the ASCII value for its uppercase counterpart when the given character is lower-case. If the given character is upper-case already, zds_toupper leaves it as it is.

Example

int returni;

char string1[512];

zds_getstring(1,"\nEnter a lower case character: ",string1);

returni=zds_toupper(string1[0]);

zds_printf("\nzds_toupper returned %i",returni);

zds_unload_dialog

int zds_unload_dialog(int nDialogFileID);

Unloads the specified DCL file.

Unloads the DCL file associated with nDialogFileID (which was obtained from a previous zds_new_dialog() call).

The zds_unload_dialog() function always returns RTNORM.

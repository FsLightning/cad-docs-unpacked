zds_vector_image

int zds_vector_image(int nXVectStart, int nYVectStart, int nXVectEnd, int nYVectEnd, int nColor);

Draws a vector in the active image.

Draws a vector in the currently active image (opened by the zds_start_image() function) from the point (nXVectStart, nYVectStart) to (nXVectEnd, nYVectEnd). The nColor parameter is an ZWCAD color number or one of the logical color numbers.

Symbolic names for the nColor attribute:

Color number ADI mnemonic Description
-2 BGLCOLOR Current background of the ZWCAD graphics screen
-15 DBGLCOLOR Current dialog box background color
-16 DFGLCOLOR Current dialog box foreground color (for text)
-18 LINELCOLOR Current dialog box line color

The origin (0,0) is the upper-left corner of the image. You can obtain the coordinates of the lower-right corner by calling the dimension function zds_dimensions_tile()).

The zds_vector_image function returns RTNORM if successful; otherwise, it returns an error code.

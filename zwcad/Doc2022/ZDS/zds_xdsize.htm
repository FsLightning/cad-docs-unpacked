zds_xdsize

int zds_xdsize (const struct zds_resbuf *entity-data, long *memused);

Determine how much space this xdata takes up.

This function returns the size (in bytes) that extended entity data takes up in an object.

The argument entity-data is a result buffer containing only the extended entity data for the object in question.

The argument memused is the amount of memory, in bytes, used by the extended entity data.

This function returns RTNORM or an error code in system variable ERRNO.

Example

zds_name last;

zds_entlast(last);

zds_resbuf * rbApps = zds_buildlist(RTSTR, "*", RTNONE);

zds_resbuf * rbData = zds_entgetx(last, rbApps);

zds_resbuf * rbXData = entitem(rbData, -3);

long lUsed;

int rc = zds_xdsize(rbXData, &lUsed);

switch (rc) {

case RTERROR:

zds_printf("\nThe last entity does not have any XData.");

break;

case RTNORM:

zds_printf("\nThe XData in the last entity takes up %li bytes.", lUsed);

break;

default:

zds_printf("\nzds_xdsize returned %i.", rc);

break;

}

zds_relrb(rbData);

rbData = NULL;

zds_relrb(rbApps);

rbApps = NULL;

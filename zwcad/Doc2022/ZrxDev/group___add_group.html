Adding a Group to the Group Dictionary
ZRX Introductory Concepts » Database Primer » Creating Objects in ZRX

The following code creates a group and then add it into the Group Dictionary.

void DBPrimer::CreateGroup(AcDbObjectId objId)
{
AcDbGroup *pGroup = new AcDbGroup();
pGroup->setName(_T("NewGroup"));
pGroup->append(objId);
AcDbDictionary *pGroupDict;
acdbHostApplicationServices()->workingDatabase()->getGroupDictionary(pGroupDict,AcDb::kForWrite);
AcDbObjectId pGroupId;
pGroupDict->setAt(_T("NewGroup"),pGroup,pGroupId);
pGroupDict->close();
pGroup->close();
}

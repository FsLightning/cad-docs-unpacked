Using CAD UI Class
Basic Interaction with ZWCAD » MFC Support in ZRX

CAD platform has a series of classes similar to the MFC user interface that allows users to quickly create the desired interface.
The interface features and displays are consistent with the CAD platform.
This example shows: how to use platform class to create a tool pallete which can be docked into the CAD.

1.Add a new class "CADDlg" which derived from CAcUiDialog for the dialog.
This is a "BUTTON2" button in the dialog. Click the button, then wait for user to get a piont and print the piont information.

class CADDlg : public CAcUiDialog
{
public:
CADDlg(CWnd* pParent = NULL);
enum { IDD = IDD_DIALOG2 };
protected:
virtual void DoDataExchange(CDataExchange* pDX);
DECLARE_MESSAGE_MAP()
public:
afx_msg void OnBnClickedButton2();
};
CADDlg::CADDlg(CWnd* pParent ) : CAcUiDialog(CADDlg::IDD, pParent)
{
}
void CADDlg::DoDataExchange(CDataExchange* pDX)
{
CDialog::DoDataExchange(pDX);
}
BEGIN_MESSAGE_MAP(CADDlg, CAcUiDialog)
ON_BN_CLICKED(IDC_BUTTON2, &CADDlg::OnBnClickedButton2)
END_MESSAGE_MAP()
void CADDlg::OnBnClickedButton2()
{
ads_point point;
if (acedGetPoint(NULL, _T("Get a point: \n"), point) == RTNORM)
{
acutPrintf(_T("Point(%g, %g, %g)\n"), point[X], point[Y], point[Z]);
}
}

2.Add a new class "CADPalette" which derived from CAdUiPalette.
Add a CADDlg variable to the CADPalette class definition, in order to show the custom dialog box in the palette.
Create the dialog box and set the size when the palette is created at "OnCreate".
When resizing the palette, resize the dialog box at the same time at "OnSize".

class CADPalette : public CAdUiPalette
{
DECLARE_DYNAMIC (CADPalette)
public:
CADPalette (HINSTANCE hInstance =NULL) ;
virtual ~CADPalette () ;
protected:
afx_msg int OnCreate (LPCREATESTRUCT lpCreateStruct) ;
afx_msg void OnSize (UINT nType, int cx, int cy);
DECLARE_MESSAGE_MAP()
private:
CADDlg PDlg;
} ;
IMPLEMENT_DYNAMIC (CADPalette, CAdUiPalette)
BEGIN_MESSAGE_MAP(CADPalette, CAdUiPalette)
ON_WM_CREATE()
ON_WM_SIZE()
END_MESSAGE_MAP()
CADPalette::CADPalette (HINSTANCE hInstance) : CAdUiPalette (), PDlg(NULL)
{
}
CADPalette::~CADPalette ()
{
}
int CADPalette::OnCreate (LPCREATESTRUCT lpCreateStruct)
{
if ( CAdUiPalette::OnCreate (lpCreateStruct) == -1 )
return (-1) ;
PDlg.Create (IDD_DIALOG2, this) ;
CRect rect ;
GetClientRect (&rect) ;
PDlg.ShowWindow(SW_SHOW);
PDlg.MoveWindow (0, 0, rect.Width (), rect.Height (), TRUE) ;
return 0 ;
}
void CADPalette::OnSize (UINT nType, int cx, int cy)
{
CWnd::OnSize(nType, cx, cy);
CRect rect;
GetClientRect(&rect);
PDlg.MoveWindow(&rect);
}

3.In the implementation function of the "CADPaletteSet" command, add and display the paletteset.
Add a new palette to the paletteset to show the custom dialog.
Finally create a function "DelPaletteSet" to delete the paletteset, when the program is exit.

CAdUiPaletteSet* pPaletteSet = NULL;
CAdUiPalette* pPalette = NULL;
void CMFCSupport::CADPaletteSet()
{
if (!pPaletteSet)
{
pPaletteSet = new CAdUiPaletteSet;
CRect rect(0, 0, 200, 200);
pPaletteSet->Create(_T("DRAW TOOL"), WS_VISIBLE | WS_CHILD | WS_CLIPCHILDREN, rect, acedGetAcadFrame());
pPalette = new CADPalette();
pPaletteSet->AddPalette(pPalette);
pPaletteSet->EnableDocking(CBRS_ALIGN_LEFT|CBRS_FLOATING);
pPaletteSet->RestoreControlBar();
}
acedGetAcadFrame()->ShowControlBar(pPaletteSet, TRUE, FALSE);
}
void CMFCSupport::DelPaletteSet()
{
if (pPaletteSet != NULL)
{
pPaletteSet->DestroyWindow();
delete pPaletteSet;
pPaletteSet = NULL;
}
if (pPalette != NULL)
{
pPalette->DestroyWindow();
delete pPalette;
pPalette = NULL;
}
}

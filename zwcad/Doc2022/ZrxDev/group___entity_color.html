Entity Color
ZRX Introductory Concepts » Entities » Modifiy Entity

The following example demonstrates change an entity Color. tips: colorIndex value is 0 Specifies BYBLOCK,265 Specifies BYLAYER,if colorIndex between 0 and 265 means different color.

void setEntityColor(AcDbObjectId entId , int colorIndex)
{
AcDbEntity* pEnt;
if (acdbOpenObject(pEnt,entId,AcDb::kForWrite) == Acad::eOk)
{
pEnt->setColorIndex(colorIndex);
pEnt->close();
}
}

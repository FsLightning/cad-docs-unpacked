Interactive Output
Basic Interaction with ZWCAD » Global Functions for Interacting with ZWCAD » Display Control

The acutPrompt() function displays a piece of information at the command line. Returns the RTNORM if the acutPrompt() function call succeeds, otherwise returns an error code. The function argument is a pointer to a string constant, and the function can not change the contents of the string.

The acutPrintf() function is equivalent to the standard C library function printf(), except that the acutPrintf() function displays the output on the text screen. The acutPrintf() function saves the string to be output in an internal buffer before sending the string to ZWCAD. The buffer contains 133 bytes, so calling the acutPrintf() function displays no more than 132 characters.

void InteractGlobalFunction::InteractOutput()
{
acedPrompt(_T("Prompt user message\n"));
acutPrintf(_T("Print user message\n"));
}

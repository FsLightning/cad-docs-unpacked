Object Snap
Basic Interaction with ZWCAD » Global Functions for Interacting with ZWCAD » General Access

Use the acedOsnap () function to specify a snap pattern to capture a point


The following code uses the acedOsnap () function to capture a circle's center.
The result is the circle's center(10.000000,10.000000,0.000000)

void InteractGlobalFunction::ObjectSnap()
{
ads_point ptCenter = {10, 10, 0};
ads_real radius = 10.0;
acedCommand(RTSTR, _T("circle"),
RT3DPOINT, ptCenter,
RTREAL, radius,
RTNONE);
ads_point pt1={0, 10, 0};//A point on the circle
ads_point pt2;
acedOsnap(pt1,_T("center"),pt2);
acutPrintf(_T("%f,%f,%f"),pt2[0],pt2[1],pt2[2]);
}

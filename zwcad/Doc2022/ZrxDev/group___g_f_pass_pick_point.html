Passing Pick Points to ZWCAD Commands
Basic Interaction with ZWCAD » Global Functions for Interacting with ZWCAD » General Access

For some commands such as Trim, Extend, and Fillet, they require the user to specify the pick point and the entity for the command. To pass such entity and point data through the function acedCommand (), the user can place the entity and the point on the result Type code between RTLB and RTLE.

void InteractGlobalFunction::CmdPassPickPoint()
{
ads_point p1={8,0,0};
ads_point p2={20,10,0};
ads_name entres;
acedCommand(RTSTR,"LINE",RTSTR,"1,0",RTSTR,"30,0",RTSTR,"",0);
acdbEntNext(NULL,entres);
acedCommand(RTSTR,_T("break"),
RTLB,RTENAME,entres,RT3DPOINT,p1,RTLE,
RT3DPOINT,p2,
0);
}

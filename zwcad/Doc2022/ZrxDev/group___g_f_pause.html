Pausing for User Input
Basic Interaction with ZWCAD » Global Functions for Interacting with ZWCAD » General Access

By acedCommand () or acedCmd () command is invoked, if there PAUSE symbol in the sequence of parameters, ZWCAD command encounters this symbol in the implementation process will pause and wait for user input information (including drag).
Pause symbol PAUSE consists of a string that contains a backslash

#define PAUSE ZCRX_T("\\")


The following code call the Rotate command to wait for the user to enter the base point and the rotation angle.
[RTSTR, ""] on behalf of the input carriage return.
[RTSTR, PAUSE] stands for pause and waiting for user input.

void InteractGlobalFunction::CmdPAUSE()
{
ads_name entres;
ads_point ptres;
acedEntSel(NULL, entres, ptres);
acedCommand(RTSTR, _T("rotate"), RTENAME, entres, RTSTR,"", RTSTR, PAUSE, RTREAL, 90.0, 0);
}

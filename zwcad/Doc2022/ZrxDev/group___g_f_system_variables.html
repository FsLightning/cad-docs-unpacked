System Variables
Basic Interaction with ZWCAD » Global Functions for Interacting with ZWCAD » General Access

Use the acedGetVar() and acedSetVar() functions to get and set the value of the zwcad system variable.
The following code use acedGetVar() get the text height Variable.

void InteractGlobalFunction::GetSystemVariables()
{
struct resbuf rb1,rb2;
acedGetVar(_T("TEXTSIZE"), &rb1);
CString str;
str.Format(_T("\n%f\n"),rb1.resval.rreal);
acutPrintf(str);
acedGetVar(_T("TEXTSTYLE"), &rb2);
if (rb2.resval.rstring != NULL)
{
acutPrintf(rb2.resval.rstring);
// Release memory acquired for string
free(rb2.resval.rstring);
}
}

The following code use acedSetVar() get the text height Variable.

void InteractGlobalFunction::SetSystemVariables()
{
struct resbuf rb;
rb.restype = RTREAL;
rb.resval.rreal = 5;
if(acedSetVar(_T("TEXTSIZE"), &rb)==RTNORM)
{
acutPrintf(_T("Set textsize success."));
}
}

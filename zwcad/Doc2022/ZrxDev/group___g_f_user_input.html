User-Input Functions
Basic Interaction with ZWCAD » Global Functions for Interacting with ZWCAD » Getting User Input

ZWCAD provides a number of global functions that prompt user input, including acedGetString, acedGetPoint, acedGetInt, acedGetKword, and acedGetReal, etc.
- acedGetInt: pauses the program execution, waiting for the user to enter an integer.
- acedGetReal: pauses the program execution, waiting for the user to enter a real number.
- acedGetString: suspend the program execution, waiting for the user to enter a string, the function is defined as:

   int acedGetString (int cronly, cons t char * prompt, char * result);


The argument cronly shows whether the string contains a space.0 means that the string is not allowed to contain spaces, and 1 means that the string can contain spaces.
- acedGetDist: pause the program execution, waiting for the user to enter a straight line distance.
- acedGetCorner: pause the program execution, waiting for the user to enter the diagonal point of the rectangular box of the graphics window.
- acedGetAngle: pauses the program, waiting for the user to enter an angle relative to the current value set by the ANGBASE system variable.
acedInitGet, acedGetKword, and acedGetInput are used to prompt the user to enter a keyword and perform different actions based on the keywords entered by the user.
The following code prompts the user to enter a point and enter a distance after successful execution

void InteractGlobalFunction::UserInput()
{
ads_point pt;
int nReturn=acedGetPoint(NULL,_T("please select a point:\n "),pt);
switch(nReturn)
{
case RTNORM:
acutPrintf(_T("Point:(%f,%f,%f)\n"),pt[X],pt[Y],pt[Z]);
acutPrintf(_T("User entered a valid value\n"));
ads_real fDist;
if (acedGetDist(pt,_T("please put distance\n"),&fDist)==RTNORM)
{
acutPrintf(_T("RealNumber:%f\n"),fDist);
}
break;
case RTERROR:
acutPrintf(_T("The function call failed\n"));
break;
case RTCAN:
acutPrintf(_T("User entered ESC\n"));
break;
case RTNONE:
acutPrintf(_T("User entered only ENTER\n"));
break;
case RTREJ:
acutPrintf(_T("Rejected the request as invalid\n"));
break;
case RTKWORD:
acutPrintf(_T("User entered a keyword or arbitrary text\n"));
break;
}
}

Wild-Card Matching
Basic Interaction with ZWCAD » Global Functions for Interacting with ZWCAD

The following code uses the acutWcMatch() function for string wildcard comparison.

The acutWcMatchEx() function has an additional argument to allow it to ignore case.

void InteractGlobalFunction::WildCardMatch()
{
if (acutWcMatch(_T("allofxzsdf"), _T("allof*")) == RTNORM)
{
acutPrintf(_T("Match success,use acutWcMatch\n"));
}
if (acutWcMatchEx(_T("ALLofxzsdf"), _T("allof*"),true))
{
acutPrintf(_T("Match success,use acutWcMatchEx\n"));
}
}

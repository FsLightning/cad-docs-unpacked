Detailed Description

You must explicitly manage the result-buffer list it creates and uses. Regardless of whether the application creates a list or passes it.

The following code shows the layer of the user selected entity, which needs to explicitly release the return result-buffer memory.

void GlobalUtilityFunction::MemoryManagement()
{
struct resbuf* rbEnt;
struct resbuf* rb;
ads_name entName;
ads_point point;
if (acedEntSel(_T("Select a Entity: \n"), entName, point) == RTNORM)
{
rbEnt=acdbEntGet(entName);
rb=rbEnt;
for(rb=rbEnt; rb!=NULL; rb=rb->rbnext)
{
if(rb->restype==8)
{
acutPrintf(_T("Ent layer:%s\n"),rb->resval.rstring);
break;
}
}
if (rbEnt!=NULL)
{
acutRelRb(rbEnt);
}
}
}

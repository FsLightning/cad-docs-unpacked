You can type in the command line: (entget (car (entsel))) To get the entity code information.

Command: (entget (car (entsel)))
Select object: ((-1 . <ENTITY NAME: 1f5a51f8>) (0 . "CIRCLE") (5 . "262") (330 . <ENTITY NAME: 1f5a1bb0>) (100 . "AcDbEntity") (67 . 0) (410 . "Model") (8 . "0") (100 . "AcDbCircle") (10 -13.1413 0.481785 0.0) (40 . 30809.6) (210 0.0 0.0 1.0))

Group code and its corresponding parameter values appear in pairs, such as (GroupGode. ParameterValues) format. E.g:
The object type group code 0 (0. "CIRCLE").
Layer group code f 8 (8. "0") .
Round group code 10 (10 -13.1413 0.481785 0.0).
Radius of the code group 40 (40.30809.6).

Entity Lists with DXF Codes in ObjectARX
Basic Interaction with ZWCAD » ZRX Global Utility Functions » Lists and Other Dynamically Allocated Data » Result-Buffer Memory Management

Use acutBuildList to construct an entity DXF list, and then use the acdbEntMake function to create an entity.

void GlobalUtilityFunction::EntMake()
{
struct resbuf* rbEnt;
ads_point point1={10,0,0};
ads_point point2={10,20,0};
rbEnt = acutBuildList(RTDXF0, _T("Line"), 10, point1, 11, point2,62, 1, RTNONE);
if (rbEnt != NULL)
{
if (acdbEntMake(rbEnt) != RTNORM)
{
acdbFail(_T("acdbEntMake fail."));
}
acutRelRb(rbEnt);
}
}

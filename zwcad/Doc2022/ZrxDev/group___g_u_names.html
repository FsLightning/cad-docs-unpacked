Entity and Selection Set Names
Basic Interaction with ZWCAD » ZRX Global Utility Functions » Variables, Types, and Values Defined in ObjectARX » General Types and Definitions

ZWCAD the names of entities and selection sets are defined as the following array type:

typedef __w64 long zds_name[2];

The following code copy an selection set name by calling the ads_name_set() function;

void GlobalUtilityFunction::adsName(void)
{
ads_name ss;
ads_name snew;
ads_point rpt;
int rt=acedSSGet(NULL,NULL,NULL,NULL, ss);
if (rt == RTNORM)
{
acutPrintf(_T("\n ss:%lx"), ss[0]);
ads_name_set(ss, snew);
if (!ads_name_nil(snew))
{
acutPrintf(_T("\n snew copies the value of ss:%lx"), ss[0]);
ads_name_clear(ss);
acutPrintf(_T("\n clear ss:%lx"), ss[0]);
}
}
acedSSFree(ss);
}

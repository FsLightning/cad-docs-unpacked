Result Buffers
Basic Interaction with ZWCAD » ZRX Global Utility Functions » Variables, Types, and Values Defined in ObjectARX » Result Buffers and Type Codes

The resulting buffer (resbuf) is a structure, which is defined as

struct resbuf {
struct resbuf *rbnext;
short restype;
union zds_u_val resval;
};

The restype variable is used to specify the type of variable saved in the union ads_u_val.

Two global functions associated with the result buffer: 
- acutRelRb: Releases the storage space of the resulting buffer list. 
- acutNewRb: Creates a new result buffer and allocates storage space for it. Use the assigned function, The storage space must be manually released.

void GlobalUtilityFunction::ReBuf()
{
struct resbuf *result;
result = acutNewRb(RTSTR);
result->resval.rstring=NULL;
int rc = acedGetFileD(_T("Select file"),
NULL,
_T("dwg;dxf"),
0,
result);
if (rc == RTNORM)
acedAlert(result->resval.rstring);
acutRelRb(result);
}

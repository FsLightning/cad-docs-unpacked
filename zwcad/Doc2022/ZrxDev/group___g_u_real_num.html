Real Numbers And Points
Basic Interaction with ZWCAD » ZRX Global Utility Functions » Variables, Types, and Values Defined in ObjectARX » General Types and Definitions

ZWCAD real values are defined as the following double type:

typedef double ads_real;

ZWCAD points are defined as the following array type:

typedef ads_real ads_point[3];

The following code show how to use ads_point;

void GlobalUtilityFunction::use_ads_point()
{
ads_point pt1={10,10,0};
ads_point pt2={10.23,6.88,0};
acutPrintf(_T("pt1 point:(%f,%f,%f)\n"),pt1[X],pt1[Y],pt1[Z]);
pt1[X]=pt2[X];
pt1[Y]=pt2[Y];
pt1[Z]=pt2[Z];
acutPrintf(_T("pt1 copies the value of pt2:(%f,%f,%f)\n"),pt1[X],pt1[Y],pt1[Z]);
ads_point pt3;
ads_point_set(pt2, pt3);
acutPrintf(_T("pt3 copies the value of pt2:(%f,%f,%f)\n"),pt3[X],pt3[Y],pt3[Z]);
}

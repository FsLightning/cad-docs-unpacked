Function Return Values versus Function Results
Basic Interaction with ZWCAD » ZRX Global Utility Functions » Common Characteristics of ZRX Library Functions

The following code shows the return value of the acedGetPoint () function and prints the result value if a point is successfully selected.

void GlobalUtilityFunction::ReturnValues()
{
ads_point pt;
int nReturn=acedGetPoint(NULL,_T("please select a point:\n "),pt);
switch(nReturn)
{
case RTNORM:
acutPrintf(_T("Point:(%f,%f,%f)\n"),pt[X],pt[Y],pt[Z]);
acutPrintf(_T("User entered a valid value\n"));
break;
case RTERROR:
acutPrintf(_T("The function call failed\n"));
break;
case RTCAN:
acutPrintf(_T("User entered ESC\n"));
break;
case RTNONE:
acutPrintf(_T("User entered only ENTER\n"));
break;
case RTREJ:
acutPrintf(_T("Rejected the request as invalid\n"));
break;
case RTKWORD:
acutPrintf(_T("User entered a keyword or arbitrary text\n"));
break;
}
}

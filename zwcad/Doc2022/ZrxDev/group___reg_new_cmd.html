Registering new commands
ZRX Introductory Concepts » ZRX Application Basic

Register a new command by calling function:

Zcad::ErrorStatus addCommand(const ZTCHAR * cmdGroupName,
const ZTCHAR * cmdGlobalName,
const ZTCHAR * cmdLocalName,
ZSoft::Int32 commandFlags,
ZcRxFunctionPtr FunctionAddr,
ZcEdUIContext *UIContext=NULL,
int fcode=-1,
HINSTANCE hResourceHandle=NULL,
ZcEdCommand** cmdPtrRet=NULL)

You must register a command in function:

void initApp();

see Create a ZRX Project

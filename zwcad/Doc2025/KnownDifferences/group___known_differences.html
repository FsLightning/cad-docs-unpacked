Known Differences to ObjectARX
Known differences to ObjectARX And Known Error ? Module Differences ? ObjectARX Reference Differences
extern
Starting from ZWCAD2017 "extern "c" function()" changed to "extern function()"
For example:
AUTOCAD2019: extern "C" HWND adsw_acadMainWnd();
ZWCAD2024: extern HWND adsw_acadMainWnd();
.dbx
ObjectArx design custom entities usually use .dbx extension file to which stands for database extension.
ZRX does not have a .dbx extension file but can be replaced with .zrx.
Plotter Configuration File.
It should be noted that AUTOCAD plotter configuration suffix is ". Pc3", and ZWCAD plotter configuration suffix is ". Pc5".
For example:
AUTOCAD2019: "DWG to PDF.pc3 "
ZWCAD2024: "DWG to PDF.pc5 "
AcRxClass
ZWCAD2024 and AUTOCAD2019 differ from ZWCAD2017:
ZWCAD2024 and AUTOCAD2018:AcRxClass constructors and destructors are protected,So if you want to save AcRxClass in the container, you can only save the pointer.
For example:
ZWCAD2024 and AUTOCAD2019:
std::vector<AcRxClass*> vec;
Int CompareAcRxClass(AcRxClass *const *p1, const void *p2)
{
const AcRxClass **pp2 = (const AcRxClass**)p2;
if (p1==pp2)
return 0;
return 1;
}
ZWCAD+2015 and ZWCAD2017:
std::vector<AcRxClass> vec;
Int CompareAcRxClass(const AcRxClass *p1, const void *p2)
{
const AcRxClass *pp2 = (const AcRxClass*)p2;
if (p1==pp2)
return 0;
return 1;
}

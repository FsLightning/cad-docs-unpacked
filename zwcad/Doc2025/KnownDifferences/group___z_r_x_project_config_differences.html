ZWSoft ZRX for ZWCAD: Known Differences  2024
ZRX Project Config Differences
Known differences to ObjectARX And Known Error » Project Config Differences
Developing Environment
ZWCAD2024: Visual Studio 2017
AUTOCAD2019: Visual Studio 2017
Header Files
In project property dialog: [C/C++ -> General -> Additional Include Directories]
For example:
AUTOCAD2019: .\Autodesk_ObjectARX_2019_Win_64_and_32_Bit\inc;.\Autodesk_ObjectARX_2019_Win_64_and_32_Bit\inc-x64
ZWCAD2024: .\ZWCAD_2024_ZRXSDK\arxport;.\ZWCAD_2024_ZRXSDK\inc
Library Paths
In project property dialog: [Linker -> General -> Additional Library Directories]
For example:
AUTOCAD2019: .\Autodesk_ObjectARX_2019_Win_64_and_32_Bit\lib-x64
ZWCAD2024: .\ZWCAD_2024_ZRXSDK\lib-x64
Output File
In project property dialog: [Linker -> General -> Output File]
For example:
AUTOCAD2019: **.arx
ZWCAD2024: **.zrx
Additional Dependencies
In project property dialog: [Linker -> Input -> Additional Dependencies]:
For example:
AUTOCAD2019: rxapi.lib;acge23.lib;acdb23.lib;acad.lib; etc
ZWCAD2024: ZwRx.lib; ZwZrx.lib; ZwDatabase.lib; ZwGeometry.lib; ZWCAD.lib; ZdUI.lib; ZwUI.lib; ZwDatabaseMgd.lib; ZwAuto.lib; ZwBase.lib; ZwGs.lib; ZwPAL.lib; ZwTc.lib;
Debug
In project property dialog: [Debug -> Command]:
For example:
AUTOCAD2019: .\Autodesk\AutoCAD 2019\acad.exe
ZWCAD2024: .\ZWSOFT\ZWCAD 2024\ZWCAD.exe
The .def File
If there is not this file in your project, this step can be skipped,
but please make sure that acrxEntryPoint was exported and you needed to change it to zcrxEntryPoint
For example:
AUTOCAD2019:
EXPORTS
acrxEntryPoint PRIVATE
acrxGetApiVersion PRIVATE
ZWCAD2024:
EXPORTS
zcrxEntryPoint PRIVATE
zcrxGetApiVersion PRIVATE
Entry Point:zcrxEntryPoint
AUTOCAD2019: extern "C" __declspec(dllexport) AcRx::AppRetCode acrxEntryPoint(AcRx::AppMsgCode msg, void *p)
ZWCAD2024: extern "C" __declspec(dllexport) AcRx::AppRetCode zcrxEntryPoint(AcRx::AppMsgCode msg, void *p)
Macros
ZWCAD does not support using ACDB_ REGISTER_ OBJECT_ ENTRY_ AUTO macro to register reactor.

Programming Overview of ZDS (ZWCAD Development System)

ZDS is an acronym for ZWCAD Development System, a C/C++ language interface compatible with the ADS® (AutoCAD® Development System) interface found in other CAD systems. ZDS provides hooks into ZWCAD® , allowing you to create custom applications.

ZDS communicates with the user and ZWCAD via LISP (akin to the way that ADS works with AutoLISP® and other CAD systems). However, ZDS greatly augments the capabilities found in LISP with a much more powerful command base that can also access the operating system. This allows you to create even more sophisticated custom solutions. And, because ZDS applications are compiled, they are typically faster than LISP applications.

Prefixes to ZDS Functions

If you're familiar with ADS, note that one primary difference between ADS and ZDS is that most ZDS functions have an zds_ prefix (except for special dialog-related ZDS functions, which have the zds_ prefix), whereas ADS functions have a variety of prefixes, such as ads_, acad_, and acrx_.

Functions Unique to ZDS

ZDS offers several additional capabilities beyond those available in ADS.

ZDS Functions

Function Description
zds_name_clear Clears the entity name or selection set.
zds_name_equal Checks if two entity names or selection sets are equal.
zds_name_nil Checks if the entity name or selection set has been cleared.
zds_name_set Copies one entity name or selection set to another.
zds_point_set Copies a point from one variable to another.

ADS Functions Not Supported by ZDS

The following ADS functions are not supported by ZDS: ads_arxload, ads_arxloaded, ads_arxunload, all ads_dict-related functions, ads__msize, ads_namedobjdict, ads_regappx, ads_ssgetx, ads_ssnamex, ads_tablet, ads_xstrcase, ads_xstrsave, ads_ssGetKwordCallbackPtr, ads_ssGetOtherCallbackPtr, adsw_acadMainWnd, and adsw_acadDocWnd.

Selection Modes

ZWCAD supports the following selection modes:

ZWCAD selection modes

Mode Meaning
NULL (none) Single pick point
"B" Box
"C" Crossing
"CC" Crossing Circle
"CP" Crossing Polygon
"F" Fence
"I" Implied
"L" Last
"O" Outside
"OC" Outside Circle
"OP" Outside Polygon
"P" Previous
"PO" Point
"W" Window
"WC" Window Circle
"WP" Window Polygon
"X" All

Selection Modes Not Supported by ZWCAD

ZWCAD does not support the following selection modes found in other CAD systems: "G", ":$", ":?", ":D", ":E", ":K", ":N", ":S", ".", "#", "A", and "M" nor the Boolean operators <NOT>, <OR>, and <AND>.

ZDS Return Codes

ZDS functions return these request type codes:

Request Code Meaning
RSRSLT 1 Result is returned.
RSERR 3 Error in evaluation; no result is returned.
RQSAVE 14 ZWCAD is saving the drawing.
RQEND 15 ZWCAD is exiting.
RQQUIT 16 ZWCAD is quitting.
RQCFG 22 ZWCAD is returning from the Config command.
RQXLOAD 100 Load external functions.
RQXUNLD 101 Unload the application.
RQSUBR 102 External subroutine request.
RQHUP 105 ZWCAD is disconnected from the application.
RQXHELP 118 Define help for the function.
RTERROR -5001 An error has occurred.
RTCAN -5002 User canceled the command by pressing Esc.
RTREJ -5003 ZWCAD rejected the request, probably due to invalid format.
RTFAIL -5004 Link failure between ZDS and ZWCAD.
RTKWORD -5005 Keyword returned from getxxx routine.
RTNORM 5100 Request succeeded.

zds_cmd

int zds_cmd (const struct zds_resbuf *cmdlist);

Execute an ZWCAD® command.

This function executes one or more ZWCAD commands. It supplies the input for an ZWCAD command from a linked list, as if it were being entered at the command line.

The portion const struct zds_resbuf *cmdlist represents a linked list that includes all the input required to complete the given command(s).

This function returns one of the following: RTNORM (the function executed normally), RTCAN (the user canceled the command), RTREJ (the command sequence was rejected by ZWCAD), or RTERROR (some other error).

Example

struct zds_resbuf *listbuf;

zds_point point1 = {2.0, 2.0};

zds_point point2 = {3.0, 3.0};

listbuf=zds_buildlist(RTSTR,"circle",RTPOINT,point1,RTPOINT,point2,RTNONE;

zds_cmd(listbuf);Draws a circle given points as variables.

zds_relrb(listbuf);Releases the result buffer.

listbuf=zds_buildlist(RTSTR,"circle",RTSTR,"5,2",RTSTR,"6,3",RTNONE);

zds_cmd(listbuf); Draws a circle given points as strings.

zds_relrb(listbuf); Releases the result buffer.

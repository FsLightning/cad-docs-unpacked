zds_dimensions_tile

int zds_dimensions_tile(zds_hdlg hDialog, char *szControlKey, short *nWidthLess1, short *nHeightLess1);

Retrieves the dimensions of a tile in dialog box units.

Use zds_dimensions_tile() with the following functions that require you to specify absolute tile coordinates:zds_vector_image(), zds_fill_image(), zds_slide_image().

The szControlKey argument, which is case sensitive, specifies the tile. The coordinates returned are the maximum allowed within the tile; coordinates are 0. Because based, these functions return one less than the total X or Y direction (X-1 and Y-1). The width of the tile is returned in the nWidthLess1 argument, and the height is returned in nHeightLess1.The hDialog argument specifies the dialog box.

This function returns RTNORM if successful; otherwise, it returns an error code.

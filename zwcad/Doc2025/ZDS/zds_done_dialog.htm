zds_done_dialog

int zds_done_dialog(zds_hdlg hDialog,long nMessage);

Terminates the current dialog box and stops displaying it. Must be called from within a callback function.

zds_done_dialog() must be called from within a callback function (see zds_action_tile()).

Note If you provide a callback for the button whose key is "accept" or "cancel" (usually the OK and Cancel buttons, though the labels can vary), the callback must call zds_done_dialog() explicitly. If it doesn't, the cursor can be frozen in the dialog box. If you don't provide an explicit callback for these buttons and instead use the standard exit buttons, ZWCAD handles them automatically.

The nMessage argument is not optional. It must be set to DLGOK for OK, DLGCANCEL for Cancel, or some other positive integer value (nMessage >= DLGSTATUS) whose meaning depends on your application. The hDialog argument specifies the dialog box.

The function returns RTNORM if successful; otherwise, it returns an error code.

zds_done_positioned_dialog

int zds_done_positioned_dialog(zds_hdlg hDialog, int nMessage, int *nXPos, int *nYPos);

Resembles zds_done_dialog() but also returns the current (X,Y) position of the dialog box.

Functions the same as zds_done_dialog() but also returns the (X,Y) location of the dialog box in the nXPos and nYPos arguments. On platforms that enable the user to move the dialog box, your application can save these coordinates and use them in a subsequent call to zds_new_positioned_dialog().

Except for the coordinates, zds_done_positioned_dialog() behaves exactly like zds_done_dialog().

The zds_done_positioned_dialog() function returns RTNORM if successful; otherwise, it returns an error code.

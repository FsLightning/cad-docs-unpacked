zds_entdel

int zds_entdel (const zds_name entity);

Delete (or undelete) this object.

This function deletes an object from a drawing, or, if the entity specified is already deleted, zds_entdel undeletes it. The object can be toggled between the two states (deleted and undeleted) at any point during the session. When the drawing is closed, any deleted objects are permanently erased.

The argument entity represents the object to be deleted (or undeleted).

This function returns RTNORM or an error code.

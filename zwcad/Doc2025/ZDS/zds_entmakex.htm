zds_entmakex

int zds_entmakex (const struct zds_resbuf *entity-list, zds_name new-entity);

Make a new object with these properties.

This function is identical to the zds_entmake function. Its original purpose was to provide a function that was similar to zds_entmake, except not assign an owner, which would be useful for creating an object that is not saved with the drawing. However, for compatibility reasons, zds_entmakex does assign an owner and is saved to the drawing because the corresponding ADS function does so as well.

This function returns RTNORM if it succeeds. It returns RTREJ if it cannot update the database.

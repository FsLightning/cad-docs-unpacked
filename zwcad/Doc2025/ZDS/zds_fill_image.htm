zds_fill_image

int zds_fill_image(int nULCornerX, int nULCornerY, int nWidthLess1, int nHeightLess1, int nColor);

Draws a filled rectangle in the currently active image.

Draws a filled rectangle in the currently active image (opened by zds_start_image()). The nColor parameter is an ZWCAD color number or one of the logical color numbers described in the following table.

Symbolic names for the nColor attribute:

Color number ADI mnemonic Meaning
-2 BGLCOLOR Current background of the ZWCAD graphics screen
-15 DBGLCOLOR Current dialog box background color
-16 DFGLCOLOR Current dialog box foreground color (for text)
-18 LINELCOLOR Current dialog box line color

The first (upper-left) corner of the rectangle is located at (nULCornerX, nULCornerY), and the second (lower-right) corner is located at the relative distance (nWidthLess1, nHeightLess1) from the first (nWidthLess1and nHeightLess1must be positive values). The origin (0,0) is the upper-left corner of the image. You can obtain the coordinates of the lower-right corner by calling the dimension function zds_dimensions_tile().

The zds_fill_image function returns RTNORM if successful; otherwise, it returns an error code.

zds_get_tile

int zds_get_tile(zds_hdlg hDialog,char *szControlKey,char *szControlInfo,long lInfoMaxLength);

Retrieves the current run-time value of a tile.

The szControlKey argument, which is case sensitive, specifies the tile. The szControlInfo argument is set to the value. The application must allocate space for the szControlInfo string and pass its length in the lInfoMaxLength argument. The lInfoMaxLength argument should include room for the string terminator (\0 or EOS). The hDialog argument specifies the dialog box. The zds_get_tile() function returns RTNORM if it succeeds; it returns RTERROR if it can't find the specified tile.

If the tile is a list box (or a pop-up list) and no item is selected,zds_get_tile() returns the empty string ("").

The function returns RTNORM if successful; otherwise, it returns an error code.

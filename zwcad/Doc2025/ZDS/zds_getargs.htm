zds_getargs

struct zds_resbuf *zds_getargs (void);

Return the arguments to the current ZDS application.

This function returns the arguments to the specified external function and returns a pointer to a linked list of arguments. When there are no arguments, this function returns a null pointer. There is no need to call zds_relrb since this function automatically releases the list.

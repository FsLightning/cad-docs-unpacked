zds_getfuncode

int zds_getfuncode (void);

Determine the code number of that application.

This function returns the function code number for an ZDS command typed at the command line. ZWCAD® associates a function code with the external function name during the call made by zds_defun. The function zds_getfuncode does not accept any arguments.

When successful, it returns the function code; otherwise, it returns RTERROR.

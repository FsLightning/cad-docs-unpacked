zds_handent

int zds_handent (const char *enthandle, zds_name entname);

Determine the entity name of this object.

This function returns the entity name, given a pointer to the handle for a specific object.

The argument enthandle represents the handle for the object whose name you wish to retrieve.

The argument entname is the name of the object with the given handle.

This function returns RTNORM or RTERROR and sets an error code.

Example

struct zds_resbuf *entlist;

struct zds_resbuf *templist;

zds_name entname1, entname2;

char string1[512];

zds_point pt;

zds_entsel ("Pick an entity: ", entname1, pt);

entlist=zds_entget(entname1); // Get the resbuf for that object.

// Go through the object's resbuf...

for (templist=entlist;templist->rbnext!=NULL;templist=templist->rbnext)

 {

   if (templist->restype==5)

   {// Restype=5 is the handle

     zds_handent(templist->resval.rstring,entname2);

     zds_getstring(0,"Press Enter to highlight the object. ",string1);

     zds_redraw(entname2,3);

   }

 }

zds_relrb(entlist);

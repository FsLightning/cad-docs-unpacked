zds_init

int zds_init (int argc, char *argv[ ... ]);

Initialize the ZDS app.

The is the first function called in any ZDS application.

The arguments argc and argv are the same arguments passed to the main() function of the ZDS application.

This function returns either any value. When the ZDS application does not work, this function returns control to the command line.

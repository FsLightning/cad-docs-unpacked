zds_islower

int zds_islower (int val);

Determine whether this a lower-case character.

This command determines whether a given character is lower-case (a-z). The argument val is the ASCII value for a character.

This function returns non-zero if the given character is alphabetical. Otherwise, it returns zero.

Example

int returni;

char string1[512];

zds_getstring(1,"\nEnter a lowercase character: ",string1);

returni=zds_islower(string1[0]);

if (returni == 0)

zds_printf("\nThe character is NOT lowercase. ");

else

zds_printf("\nThe character IS lowercase. ");

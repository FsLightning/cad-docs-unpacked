zds_load_dialog

int zds_load_dialog(char *szFileToLoad,int *nDialogFileID);

Loads a DCL file (which can contain multiple dialog boxes).

The szFileToLoad argument is a string that specifies which DCL file to load. You don't have to include the .dcl file-name extension in the dclfile argument. The function returns an integer value (nDialogFileID) used as a handle in subsequent calls to the zds_new_dialog() and zds_unload_dialog() functions. This function is the complement of the zds_unload_dialog() function. An application can load multiple DCL files by multiple zds_load_dialog() calls.

The zds_load_dialog() function searches for files according to the current ZWCAD library search path, in the following order:

  1. The current directory

  2. The directory containing the current drawing file

  3. The directories named by the ZWCAD environment variable (if this variable has been specified)

  4. The directory containing the ZWCAD program files

This is the path that ZWCAD uses to search for menu and other support files.

The zds_load_dialog() function returns RTNORM if successful; if it cannot open the file, it returns RTERROR.

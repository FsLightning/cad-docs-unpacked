zds_mode_tile

int zds_mode_tile(zds_hdlg hDialog, char *szControlKey, int nTileState);

Sets a tile's mode.

The szControlKey argument specifies the tile. The nTileState argument is an integer value whose meaning is summarized in the following table.

Tile nTileState codes for zds_mode_tile:

Value ADSRX symbol Description
0 MODE_ENABLE Enable tile
1 MODE_DISABLE Disable tile
2 MODE_SETFOCUS Set focus to tile
3 MODE_SETSEL Select edit box contents
4 MODE_FLIP Flip image highlighting on or off

For the nTileState, use one of the symbolic codes shown in the table. The hDialog argument specifies the dialog box.

This function returns RTNORM if successful; otherwise, it returns an error code.

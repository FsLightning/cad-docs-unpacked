zds_nentsel

int zds_nentsel (const char *prompt, zds_name entityname, zds_point point, zds_point ecs2wcs[4], struct zds_resbuf **nestblks);

Get information about this nested object.

Note Do not use this function in new programs. It is provided for backward compatibility with third-party applications written for earlier CAD programs. For new programs, use zds_nentselp.

This function works exactly like zds_nentselp except that instead of a 4 x 4 transformation matrix, zds_nentsel creates a 4 x 3 matrix stored as an array of four points.

Example

char *prompt;

zds_name entname;

zds_point entpoint;

zds_point ecs2wcs[4];

struct zds_resbuf *nestblks;

zds_nentsel("Pick your favorite object: ",entname, entpoint, ecs2wcs, &nestblks);

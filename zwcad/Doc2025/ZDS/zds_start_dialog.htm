zds_start_dialog

int zds_start_dialog(zds_hdlg hDialog, int *nMessage);

Starts a dialog box and begins accepting user input.

The dialog box must have been initialized by a previous zds_new_dialog() call. It remains active until a callback function calls zds_done_dialog(). Usually zds_done_dialog() is associated with the tile whose key is "accept" (typically the OK button) and the tile whose key is "cancel" (typically the Cancel button).

The hDialog argument specifies the dialog box. The nMessage argument is set to the value of the nMessage argument passed to zds_done_dialog(). Normally, nMessage is set to one of the following: DLGOK if the user pressed OK, DLGCANCEL if the user pressed Cancel, or DLGALLDONE if all dialog boxes were terminated by a call to zds_term_dialog().

However, if zds_done_dialog() is passed a nMessage >= DLGSTATUS, dlg _start_dialog() sets the nMessage argument to this value, whose meaning depends on the application.

The function returns RTNORM if successful; otherwise, it returns an error code.

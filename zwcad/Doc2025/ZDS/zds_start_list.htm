zds_start_list

int zds_start_list(zds_hdlg hDialog,char *szControlKey, int nChangeAddNew,int nWhichItem);

Starts processing the specified list box or pop-up list.

Starts processing a list in the list box or pop-up list tile specified by szControlKey. Subsequent calls to zds_add_list() affect this list until the application calls zds_end_list(). The nChangeAddNew argument is one of the symbolic codes described in the following table.

List box codes for zds_start_list:

Value ZDS symbol Description
1 LIST_CHANGE Change selected list contents
2 LIST_APPEND Append new list entry
3 LIST_NEW Delete old list and create new list (in LISP, the default)

The nWhichItem argument is ignored unless the zds_start_list() call begins a change operation (LIST_CHANGE), in which case the nWhichItem indicates the list item to change by the subsequent zds_add_list() call. The nWhichItem is zero based.

The nWhichItem argument must be specified (the LIST_APPEND and LIST_NEW cases simply ignore its value). The hDialog argument specifies the dialog box.

The zds_start_list() function returns RTNORM if successful; otherwise, it returns an error code.

zds_swapscreen

int zds_swapscreen (void);

Display the other window.

This function displays the other window. ZWCAD® has two windows: the drawing window and the Prompt History window. When one is covered by the other, this function displays the hidden window.

The zds_swapscreen function is unique to ZWCAD.

The function has no arguments.

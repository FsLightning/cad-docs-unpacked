Default Option
Basic Interaction with ZWCAD » ZWCAD Command Prompt Standard » CommandLine Interface » Format for the Command Line Prompt

The following example shows Default Option.
"<>": square brackets, this parentheses is the system default or the current option to be executed, if you do not meet the user's drawing requirements, enter the new value.

Command: UCS
Currently in World UCS.
Specify origin of UCS or [?/Face/3point/Delete/OBject/Origin/Previous/Restore/Save/View/X/Y/Z/ZAxis/World] <World>: 3
Specify the new origin <0,0,0>:
Specify the point on positive X axis <1,0,0>: 10,10,0
Specify the point in X-Y plane with positive Y value <0,1,0>:

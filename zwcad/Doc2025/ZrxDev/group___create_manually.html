Create a ZRX Project Manually
ZRX Introductory Concepts » Overview of ZRX » Setting up a ZRX Project Using Microsoft Visual Studio

This section introduces how to create a ZRX project manually. Follow the steps below to set the project properties.


a) If you use MBCS in your project, please change to "Use Unicode Character Set" as below.

For Windows XP Edition, please change the Platform Toolset to "Visual Studio 2017 - Windows XP (v141_xp)" as below.


b) Add the path of "ZRXSDK\arxport". For example, the path is "C:\ZWCAD_ZRXSDK\arxport". It is shown as the figure below.

For Windows XP Edition, please change the Preprocessor Definitions, add "WINVER=0x0501;_WIN32_WINNT=0x0501;" as below.


c) Change [Treat wchart_t as Built-in Type] and [Force Conformance in For Loop Scope] to "Yes" as below.


d) Add the path of "ZRXSDK\lib", for example, "C:\ZWCAD_ZRXSDK\lib-$(Platform)". It's shown as the figure below.


e) Add follow dependency libraries: "ZwdUI.lib;ZwAuto.lib;ZWCAD.lib;ZwDatabase.lib;ZwDatabaseMgd.lib;ZwGeometry.lib;ZwRx.lib;ZwUI.lib;ZwZrx.lib;ZwPAL.lib;". Add the path of Module Definition File, for example, "ZrxProject". About the contents of the file, see the example.



An example of ZrxProject.def

LIBRARY ""
EXPORTS
zcrxEntryPoint PRIVATE
zcrxGetApiVersion PRIVATE

Erasing Objects
ZRX Introductory Concepts » Database Objects

The erase() function does not delete the entity immediately, and the deleted entity can use erase (kfalse) to undo the deletion.
The following code uses the erase() function to delete the entity selected by the user.

void DBObjects::EraseObject()
{
ads_name en;
ads_point pt;
if(acedEntSel(_T("\nSelect the entity you want to erase:"),en,pt) != RTNORM)
{
return;
}
AcDbObjectId entId;
Acad::ErrorStatus es = acdbGetObjectId(entId,en);
if (es==Acad::eOk)
{
AcDbEntity* pEnt;
acdbOpenAcDbEntity(pEnt,entId,AcDb::kForWrite);
pEnt->erase();
pEnt->close();
}
}

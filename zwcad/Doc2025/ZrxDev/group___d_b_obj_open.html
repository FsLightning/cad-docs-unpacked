Opening and Closing Database Objects
ZRX Introductory Concepts » Database Objects

You can use acdbOpenObject() to open an object.
You have three ways to open an object.
- kForRead.As long as the object is not already open for write or for notify,the obeject can be opened or read by up to 256 readers.
- kForWrite.If the object is not already open,you can open it for write.
- kForNotify.if the object is already closed,open for read,or open for write ,but not already open for notify,you can open it for notify.

AcDbEntity * DBObjects::UserOpenObject(AcDbObjectId objID,AcDb::OpenMode openMode)
{
AcDbEntity *pEnt;
Acad::ErrorStatus eStat=acdbOpenObject(pEnt,objID,openMode);
switch(eStat)
{
case Acad::eOk:
acutPrintf(_T("Succeeds."));
break;
case Acad::eAtMaxReaders:
acutPrintf(_T("ReadCount=256."));
break;
case Acad::eWasOpenForRead:
acutPrintf(_T("The Object is already open for read."));
break;
case Acad::eWasOpenForNotify:
acutPrintf(_T("The Object is open for notify."));
break;
}
return pEnt;
}


When you open an object, you should close it as early as possible.

pEnt->close();


If there is only one reader of the object,you can upgrade the open status to write.
Similarly, if your object is open for notify,you can use upgradeFromNotify to upgrade its open status to write

Use downgradeOpen() to downgrade its status to read.
Similarly,Use downgradeToNotify() to downgrade status to notify.

pEnt->upgradeOpen();
pEnt->setColorIndex(2);//Change a Entity Object Data
pEnt->downgradeOpen();

You can refer an AcDbObject object by handle,object ID or c++ instance pointer.

void DBObjects::userSelectEntity()
{
ads_name objName;
ads_point pt;
//It is returned in ads_name form ads_name is equivalent to AcDbObjectId.
acedEntSel(_T("\n User select an entity:"),objName,pt);
AcDbObjectId objID;
//Exchange ads_name to Object ID
//You can exchange Object ID to ads_name by use acdbGetAdsName(ads_name& objName,AcDbObjectId objId).
acdbGetObjectId(objID,objName);
AcDbEntity *pEnt;
AcDb::OpenMode openMode=AcDb::OpenMode::kForRead;
pEnt=UserOpenObject(objID,openMode);
if (pEnt!=NULL)
{
pEnt->upgradeOpen();
pEnt->setColorIndex(2);
pEnt->downgradeOpen();
pEnt->close();//Close the database object.
}
}

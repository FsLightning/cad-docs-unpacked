Database Overview
ZRX Introductory Concepts » Database Primer

The database structure of ZRX can be divided into four parts: Block Table, Layer Table, Other Symbol Tables and Named Object Dictionary.

Block Table
Define a pointer to a AcDbBlockTable object:

AcDbBlockTable *pBlockTable;

Obtain the Block Table for current drawing:

acdbHostApplicationServices()->workingDatabase()->getBlockTable(pBlockTable,AcDb::kForRead);

Define a pointer to a AcDbBlockTableRecord Objectï¿½?

AcDbBlockTableRecord *pBlockTableRecord;


Layer Table
Define a pointer to a AcDbLayerTable object:

AcDbLayerTable *pLayerTable;


Other Symbol Tables
Define a pointer to AcDbDimStyleTable object:

AcDbDimStyleTable *pDimStyleTable;


Named Object Dictronary
Define a pointer to AcDbGroup object:

AcDbGroup pGroup;


Note
You can call the following function to abtain the database for current drawing:
acdbHostApplicationServices()->workingDatabase()

When you are done using database objects, you must close them, for example:
pBlockTable->close();
pBlockTableRecord->close();

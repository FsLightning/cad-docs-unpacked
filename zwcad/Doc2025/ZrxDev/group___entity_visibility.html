Entity Visibility
ZRX Introductory Concepts » Entities » Modifiy Entity

The following example demonstrates an entity Visibility or unVisibility use function "setVisibility";

void EntityVisibility(AcDbObjectId entId , Visibility isVisibility)
{
AcDbEntity* pEnt;
if (acdbOpenObject(pEnt,entId,AcDb::kForWrite) == Acad::eOk)
{
pEnt->setVisibility(isVisibility);
pEnt->close();
}
}

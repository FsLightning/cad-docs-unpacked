Filter with Compare condition
Basic Interaction with ZWCAD » Selection Set, Entity, and Symbol Table Functions » Selection Set

"acedSSGet" function can use relational operators in the filter param.
This example shows: how to select the "ARC" entities whose radius is greater than 15.
Then print the number of the blocks.

void CSelEntSy::SelectCompare()
{
struct resbuf* rb;
ads_name ssname;
ads_real radius=15;
rb = acutBuildList(RTDXF0, _T("ARC"), -4, _T(">"), 40, radius, RTNONE);
acedSSGet(_T("X"), NULL, NULL, rb, ssname);
long length;
acedSSLength(ssname, &length);
acutPrintf(_T("\n ARC: %d"), length);
acedSSFree(ssname);
}

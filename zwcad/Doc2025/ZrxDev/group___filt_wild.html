Filter with Wild Card
Basic Interaction with ZWCAD » Selection Set, Entity, and Symbol Table Functions » Selection Set

"acedSSGet" function can use wild card in the filter param.
This example shows: how to select the blocks whose name have five character start with "door".
Then print the number of the blocks.

void CSelEntSy::SelectWild()
{
struct resbuf rb;
TCHAR sbuf[20]={0};
ads_name ssname;
rb.restype=2;
_tcscpy(sbuf, _T("door?"));
rb.resval.rstring = sbuf;
rb.rbnext = NULL;
acedSSGet(_T("X"), NULL, NULL, &rb, ssname);
long length;
acedSSLength(ssname, &length);
acutPrintf(_T("\n blocks: %d"), length);
acedSSFree(ssname);
}

AutoLISP Symbols
Basic Interaction with ZWCAD » Global Functions for Interacting with ZWCAD » General Access

Use acedGetSym() and acedPutSym() functions to get and set the value of the AutoLISP variable.
The following code is equivalent to lsp expressions:

(setq mystr "ZWCAD.com")
!mystr
void InteractGlobalFunction::GetAndSetLispSymbols()
{
struct resbuf *rb1;
int rc;
rb1 = acutBuildList(RTSTR, _T("ZWCAD.com"),0);
rc = acedPutSym(_T("mystr"), rb1);
if (rc==RTNORM)
{
acutPrintf(_T("PutSym mystr is %s\n"), rb1->resval.rstring);
}
acutRelRb(rb1);
struct resbuf *rb2;
rc = acedGetSym(_T("mystr"), &rb2);
if (rc == RTNORM && rb2->restype == RTSTR)
acutPrintf(_T("GetSym mystr is %s\n"), rb2->resval.rstring);
acutRelRb(rb2);
}

The following code uses acedFindFile () function to search for a file.

void InteractGlobalFunction::FileSearch()
{
const ZTCHAR *refname = _T("zwcad.dwg");
ZTCHAR fullpath[100];
if (acedFindFile(refname, fullpath) == RTNORM)
{
acutPrintf(_T("Find file: %s.\n"), fullpath);
}
}

The acedGetFileD() function uses the standard ZWCAD file dialog box to prompt the user to enter a file name.

int acedGetFileD(const char * title,
const char * default,
const char * ext,
int flags,
struct resbuf * result);

title: the title of the dialog box.
default: the default file name.
ext: the default file extension.
flags: The parameter controls the style of the dialog with a bit value.
result: The path selected by the user.


The following code uses acedFindFileD() function to call the file dialog boxs to earch for a file.

void InteractGlobalFunction::FileSearchD()
{
struct resbuf *result;
int rc, flags;
result = acutNewRb(RTSTR);
result->resval.rstring=NULL;
flags = 2;
rc = acedGetFileD(_T("Select file"),
_T("D:\\"),
_T("dwg;dxf"),
flags,
result);
if (rc == RTNORM)
acedAlert(result->resval.rstring);
acutRelRb(result);
}

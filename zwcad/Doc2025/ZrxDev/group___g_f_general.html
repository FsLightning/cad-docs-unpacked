Detailed Description

You can use the acedCommand or acedCmd function to execute an internal command, which is the same as the direct execution of the corresponding command in ZWCAD

The number of parameters of the function is variable, and the parameters are paired. The first parameter in the parameter pair indicates the type of the parameter, and the second represents its actual data. The last parameter of the parameter table must be 0 or RTNONE.

int acedCommand(int rtype, ... unnamed);


The argument to this function is a pointer to a resbuf type

int acedCmd(const struct resbuf * rbp);


The following code Use the acedCommand function to create a circle.And then user Choose an entity to do the rotation operation.
(RTPOINT, ptCenter) and (RTSTR, "10,10,0") are the same for specifying a point coordinate

void InteractGlobalFunction::UseCmd()
{
ads_point ptCenter = {10, 10, 0};
ads_real radius = 10.0;
acedCommand(RTSTR, _T("circle"),
RT3DPOINT, ptCenter,
RTREAL, radius,
RTNONE);
ads_name entres;
ads_point ptres;
acedEntSel(NULL, entres, ptres);
int iRet = 0;
ads_point ptBase = {0, 0, 0};
ads_point ptEnd = {1, 1, 0};
acedCommand(RTSTR, _T("rotate3d"), RTENAME, entres, RTSTR, _T(""), RTPOINT, ptBase, RTPOINT, ptEnd, RTREAL, 90.0, 0);
}

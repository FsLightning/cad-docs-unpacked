Geometric Utilities
Basic Interaction with ZWCAD » Global Functions for Interacting with ZWCAD » General Access

The following sample code shows some simple calls to the geometric utility functions.
acutPolar()function finds a point by means of polar coordinates.

void InteractGlobalFunction::GeometricUtl()
{
ads_point pt1={10,10,0};
ads_point pt2={0,10,0};
ads_point endpt;
ads_real rads, length;
rads = acutAngle(pt1, pt2);
acutPrintf(_T("The Angle in the XY plane of the current UCS, in radians:%f\n"),rads);
length = acutDistance(pt1, pt2);
acutPrintf(_T("The Distance:%f\n"),length);
acutPolar(pt1, rads, length, endpt);
acutPrintf(_T("The Polar point:(%f,%f,%f)\n"),endpt[X],endpt[Y],endpt[Z]);
}

Use the acedGetKword () function to get the user input keywords.
After calling a user input function, the acedGetInput () function retrieves the keyword entered by the user.The retrieved keywords are stored in a string.
The acedGetInput () function is called after the call to the user's input function,Otherwise it is meaningless and will not succeed. Even so, the call will only succeed if the user enters a function that returns the RTKWORD value.

void InteractGlobalFunction::UserInputKWord()
{
ads_real num = 0;
acedInitGet(NULL,TEXT("Z W"));
int rc=acedGetReal(_T("Input Real Number [Or put Z / W]:\n"), &num);
if ( rc== RTNORM)
{
acutPrintf(_T("Real Number:%f\n"),num);
}
else if (rc==RTKWORD)
{
TCHAR kwork[20];
if (acedGetInput(kwork)==RTNORM)
{
acutPrintf(_T("Text:%s\n"),kwork);
}
}
}

Real-World Units
Basic Interaction with ZWCAD » Global Functions for Interacting with ZWCAD » Conversions

The function acutCvUnit () is used to convert the value of a unit into another unit value. The unit is represented by a string, and the string used must be the same as defined in the file acad.unt.
Note: You can not use the function acutCvUnit () to convert unmatched units, for example, you can not convert inches into years.
The following procedure converts the specified distance into meters.

void InteractGlobalFunction::RealWorldUnits()
{
ads_real ilen=100;
ads_real mlen;
acutCvUnit (ilen, _T("inches"), _T("meters"), & mlen);
acutPrintf(_T("inches to meters : %f\n"), mlen);
}

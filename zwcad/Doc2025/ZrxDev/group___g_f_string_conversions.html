String Conversions
Basic Interaction with ZWCAD » Global Functions for Interacting with ZWCAD » Conversions

- acdbRToS () Converts the value of the ads_real type to a string

int acdbRToS(
ads_real val,
int unit,
int prec,
ACHAR * str);

Where val is realvalue to be converted; unit specifies the unit to use, and if set to -1, the value of the LUNITS system variable in the ZWCAD drawing is used as the unit of use. precstr is number of decimal places of precision to include in the string; if set to -1, uses the current value of the ZWCAD LUPREC system variable .str Returns the conversion result.

- acdbAngToS () Converts the angle to a string by format.
- acdbDisToF() Converts a real number stored in a string into a real number based on the specified unit.

int acdbDisToF(
   const char * str,
int unit,
ads_real * v);

Where str holds the string to be converted; unit specifies the unit to use, and if set to -1, the value of the LUNITS system variable in the ZWCAD drawing is used as the unit of use. v Returns the conversion result.

- acdbAngToF() Converts a angle stored in a string into a real number based on the specified unit.

void InteractGlobalFunction::StringConversions()
{
ads_real f = 100.5;
ZTCHAR strf[12];
acdbRToS(f, -1, 4, strf);
acutPrintf(_T("real value to string: %s\n"), strf);
ads_real v;
acdbDisToF(strf,-1,&v);
acutPrintf(_T("string value to real : %f\n"), v);
ads_real ang = 3.14159;
acdbAngToS(ang, -1, 0, strf);
acutPrintf(_T("agle value to string: %s\n"), strf);
acdbDisToF(strf,-1,&v);
acutPrintf(_T("string value to angle: %f\n"), v);
}

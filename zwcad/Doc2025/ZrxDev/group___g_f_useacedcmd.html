Using acedCmd()
Basic Interaction with ZWCAD » Global Functions for Interacting with ZWCAD » General Access

The argument to the acedCmd function is a pointer to a resbuf type, and the result buffers needed here can be generated by the acutBuildList function.

struct resbuf {
struct resbuf *rbnext;
short restype;
union ads_u_val resval;
};


The acedCommand function is actually a resbuf structure for the command to be executed. So the acedCmd function and the acedCommand function are equivalent.

void InteractGlobalFunction::UseCmd2()
{
struct resbuf *rb;
ads_point ptCenter = {10, 0, 0};
ads_real radius = 10;
rb = acutBuildList(RTSTR,_T("circle"),
RTPOINT, ptCenter,
RTREAL, radius,
RTNONE);
if (rb != NULL)
{
acedCmd(rb);
}
}

Argument Lists in AutoLISP and C

The following code uses the acdbEntMake() function to receive a list of variable parameters. Simulation of AutoLisp entmake function can be any input variable parameters of the function.
The acutPrintf () function is similar to the standard C library printf () function.

void GlobalUtilityFunction::ArgumentLists()
{
struct resbuf* rbEnt;
ads_point point1={10,0,0};
ads_point point2={30,00,0};
rbEnt = acutBuildList(RTDXF0, _T("Line"), 10, point1, 11, point2, RTNONE);
if (rbEnt != NULL)
{
if (acdbEntMake(rbEnt) == RTNORM)
{
acutPrintf(_T("Line:(%.2f,%.2f,%.2f),(%.2f,%.2f,%.2f)\n"),point1[X],point1[Y],point1[Z],point2[X],point2[Y],point2[Z]);
}
acutRelRb(rbEnt);
}
}

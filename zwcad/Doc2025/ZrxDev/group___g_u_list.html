Basic Interaction with ZWCAD » ZRX Global Utility Functions » Lists and Other Dynamically Allocated Data » Result-Buffer Memory Management

The acutNewRb() function can reates a new result buffer and allocates storage space for it.
The acutBuildList() function constructs a result buffer and automatically allocates memory.
Whether using acutNewRb or using acutBuildList, you need to use the acutRelRb function to release the allocated memory

void GlobalUtilityFunction::ReBuf2()
{
ads_name ss;
struct resbuf *rb1;
rb1 = acutBuildList(RTDXF0,_T("CIRCLE"),8,_T("0"),RTNONE);
int rt=acedSSGet(_T("X"),NULL,NULL,rb1, ss);
if (rt == RTNORM)
{
long length;
acedSSLength(ss, &length);
if (length>0)
{
acutPrintf(_T("select %d entities.\n"),length);
}
}
acedSSFree(ss);
}

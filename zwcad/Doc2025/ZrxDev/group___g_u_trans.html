Transformation Matrices
Basic Interaction with ZWCAD » ZRX Global Utility Functions » Variables, Types, and Values Defined in ObjectARX » General Types and Definitions

The following code show how to use transformation matrix to scale the entities;
The ident_init function initializes the transformation matrix.
The transformation matrix is a 4 Ã— 4 order array with the ads_real data type. The first three columns of the matrix determine the ratio and the rotation, and the fourth column is a translation vector. The last line of the matrix is defined as [0 0 0 1].
With the acedXformSS() function, the selection set can use the transformation matrix. Using a transformation matrix, you can change the ratio, move, rotate, or mirror the selection of the entity.

void GlobalUtilityFunction::ident_init(ads_matrix id)
{
int i, j;
for (i = 0; i <= 3; i ++)
for (j = 0; j <= 3; j ++)
id[i][j] = 0.0;
for (i = 0; i <= 3; i ++)
id[i][i] = 1.0;
}
void GlobalUtilityFunction::Transmatrices(void)
{
ads_name ss;
int rt=acedSSGet(NULL,NULL,NULL,NULL, ss);
if (rt == RTNORM)
{
long length;
acedSSLength(ss, &length);
if (length>0)
{
ads_matrix mat;
ident_init(mat);
mat[0][X] =2;
mat[1][Y] =2;
mat[2][Z] =2;
acedXformSS(ss,mat);
}
}
acedSSFree(ss);
}

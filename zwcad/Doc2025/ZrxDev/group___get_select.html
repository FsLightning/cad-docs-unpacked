Getting Selection Set
Basic Interaction with ZWCAD » Selection Set, Entity, and Symbol Table Functions » Selection Set

The selection set is an important means for CAD to interact with the user.
The selection set allows the user to select one or more graphic objects, while providing a wealth of means to select entities that meet certain condition.
The operation of selection object is implemented by the "acedSSGet" function.

int acedSSGet (const char *str, const void *pt1, const void *pt2, const struct resbuf *filter, ads_name ss)


This example shows: how to select all the entities in current database using "A" for the str param.
Then print the number of the entities.

void CSelEntSy::SelectEnt()
{
ads_name name;
acedSSGet(_T("A"), NULL, NULL, NULL, name);
long entCount;
acedSSLength(name, &entCount);
acutPrintf(_T("\n Entities: %d."), entCount);
acedSSFree(name);
}

This example shows: how to select the polyline entities in current database.
Using "X" for the str param to set up the filter condition, and using the resbuf with "LWPOLYLINE" string for the filter condition to filt.
Then print the number of the polyline entities.

void CSelEntSy::SelectPL()
{
struct resbuf rb;
TCHAR sbuf[20]={0};
ads_name ssname;
rb.restype=0;
_tcscpy(sbuf, _T("LWPOLYLINE"));
rb.resval.rstring = sbuf;
rb.rbnext = NULL;
acedSSGet(_T("X"), NULL, NULL, &rb, ssname);
long length;
acedSSLength(ssname, &length);
acutPrintf(_T("\n POLYLINE: %d"), length);
acedSSFree(ssname);
}

This example shows: how to select the entities which is in the define rectangular region in current database.
Using "W" for the str param to set up the rectangular region selection, and using the pt1, pt2 param to define the rectangular region.
Then print the number of the entities in the rectangular region.

void CSelEntSy::SelectRect()
{
ads_name ssname;
ads_point point1, point2;
point1[X]=0;point1[Y]=0;point1[Z]=0;
point2[X]=250;point2[Y]=230;point2[Z]=0;
acedSSGet(_T("W"), &point1, &point2, NULL, ssname);
long length;
acedSSLength(ssname, &length);
acutPrintf(_T("\n Entities in rectangle: %d"), length);
acedSSFree(ssname);
}

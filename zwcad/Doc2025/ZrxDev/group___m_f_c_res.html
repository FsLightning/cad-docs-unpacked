Basic Interaction with ZWCAD » MFC Support in ZRX

Resources are important for the user interface. The ZRX program can use a macro AC_IMPLEMENT_EXTENSION_MODULE to manage the resources used by the user interface.
When the DLL dynamic library is loaded, using "AttachInstance" to initialize the define resources; and when the DLL is uninstalled, using "DetachInstance" to restore the default resources.
The resource instance using "ModuleResourceInstance" is loaded as a parameter, when registering a command.

AC_IMPLEMENT_EXTENSION_MODULE(theArxDLL);
static void initApp()
{
acedRegCmds->addCommand(_T("ZRX_DLG_SAMPLE"),
_T("ZRXDLGSAMPLE"),
_T("DLGSAMPLE"),
ACRX_CMD_MODAL,
dialogCreate,
NULL,
-1,
theArxDLL.ModuleResourceInstance());
}
extern "C" int APIENTRY
DllMain(HINSTANCE hInstance, DWORD dwReason, LPVOID lpReserved)
{
// Remove this if you use lpReserved
UNREFERENCED_PARAMETER(lpReserved);
if (dwReason == DLL_PROCESS_ATTACH)
{
theArxDLL.AttachInstance(hInstance);
}
else if (dwReason == DLL_PROCESS_DETACH)
{
theArxDLL.DetachInstance();
}
return 1; // ok
}

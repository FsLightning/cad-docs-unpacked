Opening and Closing ZRX Objects
ZRX Introductory Concepts » Database Primer » Creating Objects in ZRX

The following code shows how to open a object, then modify and close.

Acad::ErrorStatus DBPrimer::ChangeColor(AcDbObjectId entId)
{
AcDbEntity *pEnt;
acdbOpenAcDbEntity(pEnt,entId,AcDb::kForWrite);
pEnt->setColorIndex(1);
pEnt->close();
return Acad::eOk;
}

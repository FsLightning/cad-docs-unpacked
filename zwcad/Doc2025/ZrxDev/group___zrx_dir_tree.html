ZRX Directory Tree
ZRX Introductory Concepts » Overview of ZRX

Introduces the directory structure of the SDK package.

arxport
The arxport directory contains the header files. The header files contains macro conversion, in order to be compatible with ObjectARX.
Doc
The Doc directory contains the help documentation about ZRX.
inc
The inc directory contains the ZRX header files.
lib-Win32
The lib-Win32 directory contains the library files for 32bit operating systems.
lib-x64
The lib-x64 directory contains the library files for 64bit operating systems.
samples
The samples directory contains some examples of ZRX application.
Tools
The Tools directory contains a ZRX wizard.
utils
The utils directory constains a library that extend ZRX.

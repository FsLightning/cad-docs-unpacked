第三步：设置Unicode字符集

中望CAD中基于Unicode字符集处理字符串，所以推荐将项目的字符集设置为Unicode字符集。

  1. 首先，将【配置属性】-【常规】页面中的字符集修改为使用Unicode字符集。

  2. 然后，将【C/C++】-【语言】页面中的将Wchar_t视为内置类型设置为是(/Zc:wchar_t)。

  3. 最后，请在【C/C++】-【预处理器】页面中确认包含了_UNICODE和UNICODE预处理器定义并且没有取消定义。
